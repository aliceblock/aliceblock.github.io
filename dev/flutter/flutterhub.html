      # FlutterHub เขียนและรันในตัวเดียวกัน

      #Dev#Flutter#Beta#Tools

      สั้นๆกับ FlutterHub (opens new window) พอดีผมเพิ่งได้สิทธิ์มาครับ ตอนนี้ตัว FlutterHub ยังเป็น Beta อยู่ในช่วงให้ทดลองเล่น
      ตัว FlutterHub โฆษณามาว่าเราสามารถสร้างโปรเจคเขียนโค้ด Flutter ได้บนเว็บ แล้วยังสามารถรันแสดงผลแบบ Web หรือบน Emulator ได้ เช่น Android, iOS (ยังไม่เปิดให้ใช้)

      # มาลองเล่นกัน

      ก่อนอื่นก็ลองสร้าง Project ก่อนครับ ซึ่งมีข้อจำกัดนิดหน่อย(สำหรับตอนนี้) ก็คือสำหรับ Basic(เริ่มต้น) สร้างได้แค่ 1 Project และสำหรับ Pro สร้างได้ 5 Project

      ทาง FlutterHub ช่วงทดลองก็สมัคร Pro ได้ฟรีโดยใช้ข้อมูลบัตรเครดิตตามนี้
