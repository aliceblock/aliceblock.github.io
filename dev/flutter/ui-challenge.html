      # แนะนำแอปสำหรับฝึกเขียน UI Flutter

      #Dev#Flutter#App#Ui

      พึ่งจะได้มีโอกาสมาแนะนำครับ ผมดูเป็นตัวอย่างอยู่สักพักแล้วล่ะ
      เป็นแอปที่มีประโยชน์มากๆที่จะเรียนรู้เทคนิคการทำ UI ในแบบต่างๆ และแน่นอนว่าฟรีครับ 😁

      # มีการทำ Animation ด้วย

      Animation ท่ายากที่ดูโค้ดก็ไม่ยากอย่างที่คิด (เหรอ? 🤣)

      # มีการอัปเดตอยู่เรื่อยๆ

      เท่าที่สังเกตดูก็อัปเดตบ่อยอยู่เหมือนกันนะครับ (แต่ไม่รู้ว่าเพิ่มอะไรเหมือนกัน)

      # ตัวอย่างโค้ด

      ที่สำคัญคือมีตัวอย่างโค้ดให้ดูด้วยครับ!!
      สามารถพลิกดูโค้ด ดูผลลัพธ์ได้ตลอดเวลาครับ แค่นี้ก็แจ่มมากๆแล้ว

      # เพิ่มเติม

      สามารถไปตามติด ติดตาม Github Repo ของแอปได้ ที่นี่ (opens new window)
      หรือใครอยากเรียนรู้ทริคเพิ่มเติมมากกว่านี้สามารถไปค้นหาใน Youtube ได้ครับ เช่น ที่นี่ (opens new window)

      ก็สั้นๆเท่านี้ล่ะครับ ใครมีแอปอะไรแนะนำผม แนะนำเพื่อนๆก็บอกได้นะครับ 😎
      ถ้าหากมีข้อสงสัยอะไรสามารถติดต่อผ่าน Facebook Page (opens new window) ของผมได้ครับ
