        # แกะกล่อง Beyblade ชุดใหญ่ไฟกะพริบ

        #Lifestyle#Beyblade#ของเล่น#แกะกล่อง

        ทรีทูวัน โก๋...ชู้ดดดดดดด

        ของเล่นในวัยเด็ก คราวนี้จัดมาเซ็ตใหญ่กันเลยครับ
        เบย์เบลดลวดลายสวยงามที่มาพร้อมกับระบบ Burst ที่เมื่อเบย์เบลดมาชนกันจนเกิดความเสียหายระดับหนึ่ง จะเกิดการ Burst หรือกระจายออกเป็นส่วนๆ

        # แบ่งออกเป็นสายๆตามนี้

        1. Attack หรือจู่โจม (เน้นการเข้าโจมตีคู่ต่อสู้)
        2. Defense หรือป้องกัน (เน้นทนการโจมตีของคู่ต่อสู้)
        3. Stamina หรือสายอดทน (เน้นหมุนนานเพื่อยืนหยัดเป็นคนสุดท้าย)
        4. Balance หรือสายสมดุล (ไม่เด่นไปทางไหนเป็นพิเศษ)

        # เปิดกล่อง Cho-Z Custom Set กันก่อนเลย

        โดยในกล่องนี้ก็มีมาให้ทุกสายเลยครับ อาจจะซื้อแล้วเอาไปผสมๆกับชุดอื่นๆก็ได้ เพราะเซ็ตนี้มี Part ให้เล่นเยอะอยู่

        Cho-Z Custom Set
        ด้านหลัง

        ตามคำบรรยายด้านหลังแล้วมันบอกว่า

        1. Cho-Z Spriggan เป็นสาย Balance เปลี่ยนโหมดหมุนซ้าย/ขวาได้ และตัว Driver หรือชั้นล่างสุดเปลี่ยนหัวได้ด้วย จะเน้นการ Burst คู่ต่อสู่
        2. Orb Egis เป็นสาย Defense จะเน้นการยืนหยัดบนสนาม รอรับการปะทะ
        3. Crash Ragnaruk เป็นสาย Attack จะเน้นการ Burst คู่ต่อสู่
        4. Bloody Longinus เป็นสาย Stamina จะเน้นการชนศัตรูออกจากสนาม เป็นเบย์หมุนซ้าย (หรือทวนเข็มนาฬิกา)
        ส่วนของ Layer หรือชิ้นส่วนชั้นบนสุด
        ประกอบเสร็จแล้วเย่ สวยงาม 😁
